🎵 AI Music Generation using Deep Learning
📌 Project Overview

This project is an AI-based Music Generator that uses deep learning (LSTM) to create new musical compositions. The model is trained on MIDI note sequences and learns musical patterns, rhythms, and structures to generate original music.

A Streamlit web application is built to allow users to generate and download AI-composed music easily.

🚀 Features
🎼 AI-based music generation using LSTM
🎹 Trained on MAESTRO MIDI dataset
🔁 Sequence prediction of musical notes
🎧 MIDI file generation and download
🌐 Simple Streamlit web interface
🛠️ Tech Stack
Python
TensorFlow / Keras
NumPy
Music21
Streamlit
